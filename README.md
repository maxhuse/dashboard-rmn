# Dashboard (built with React/MobX and Node.js)

### How to Run?

1. Install dependencies
```sh
npm install
```

2. Initialize and fill database with data
```sh
npm run init
```

3. Build client code
```sh
npm run build:production
```

4. Start server
```sh
npm run start
```
